# GDRCD Installer

**GDRCD Installer** è uno script batch progettato per semplificare l'installazione dell'ambiente di sviluppo per GDRCD su Windows e automatizza l'installazione di **GDRCD/GDRCD**, **GDRCD/stack** e dei software prerequisiti necessari per il loro funzionamento. Questo strumento automatizza l'installazione di Git, Docker e WSL, rendendo il processo rapido e senza intoppi.

## Caratteristiche

- Installazione automatizzata di **GDRCD/GDRCD** e **GDRCD/stack**.
- Installazione dei prerequisiti: Git, Docker e WSL.
- Facile da usare tramite semplice esecuzione di un file batch.
- Compatibile solo con sistemi operativi Windows.

## Stato del progetto

Attualmente questo progetto è in fase alpha e instabile. \
Ulteriori test e rifiniture per gestire eventuali edge case sono ancora necessari e ogni contributo in proposito è bene accetto!
